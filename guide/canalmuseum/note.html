讲解提纲

北京湾

  • 燕山运动在中生代侏罗纪晚期至白垩纪，八达岭花岗石，房山汉白玉。
  • 元末开凿金口新河，引卢沟河水至通州，因水量过大，威胁城池而废弃。

大运河

  • 大运河包括京杭大运河、浙东大运河和隋唐大运河三个部分，跨越2500多年，全长3200公里，通达海河、黄河、淮河、长江、钱塘江五大水系，是世界上距离最长、规模最大的运河。
  • 春秋末期，公元前486年，吴王夫差调集民工开凿邗沟，运兵运粮，北上伐齐。因途经邗城，故得名“邗沟”。
  • 清中叶后，因南北海运兴起，津浦铁路通车，其作用逐渐减弱。
  • “南水北调”东线工程主要通道之一。
  • 大运河漂来的北京城。

第一单元 江河道渠 水运初兴

  • 宫廷乐舞人物画像石，曹操

作册奂鼎

  • "太保墉燕"铭文，意为"太保召公来到燕地建设都城"。
  • 作册奂可能是当时的史官或书记员，太保召公筑城后，在燕侯宫举行祭礼，赏赐贝币给作册奂，作册奂用这些贝币铸造了礼器。
  • 作册奂器作为西周燕国的宫廷礼器，其艺术风格也体现了周文化与地方特色的融合。
  • 掌握准确的合金配比、高温熔炼技术和精密的浇铸工艺。
  • 在青铜器上铸出清晰规整的长篇铭文，需要在陶范上精细刻写文字，并控制好浇铸时的温度和速度。

吴，邗（hán）沟

  • 邗沟南起扬州以南的长江，北至淮安以北的淮河，是最早有明确记载的运河。
  • 春秋末期（公元前486年），吴王夫差为了北上伐齐，开凿邗沟。
  • 邗沟，邗城，扬州，广陵。
  • 扬州、淮安两座历史文化名城因邗沟的开凿而诞生。

魏，鸿沟

  • 战国时魏惠王十年（前361年），为了战争需要，曾两次兴工，开挖了鸿沟。
  • 鸿沟是中国古代最早沟通黄河和淮河的人工运河，以魏都大梁为中心。
  • 鸿沟西自荥阳以下引黄河水为源，向东流经中牟、开封，折而南下，入颍河通淮河，把黄河与淮河之间的济、濮、汴、睢、颍、涡、汝、泗、菏等主要河道连接起来，构成鸿沟水系。
  • 《史记·项羽本纪》：项王乃与汉约，中分天下，割鸿沟以西者为汉，鸿沟而东者为楚。

秦，灵渠

  • 灵渠联接了长江和珠江两大水系，构成了遍布华东华南的水运网。
  • 灵渠流向由东向西，将兴安县东面的海洋河（湘江源头，流向由南向北）和兴安县西面的大溶江（漓江源头，流向由北向南）相连。
  • 湘江水在分水塘经铧嘴分流和大小天坪坝引流后，约7分水流入北渠，在高塘村与湘江故道相会，约3分水流入南渠入漓江。
  • 灵渠的凿通，沟通了湘江、漓江，打通了南北水上通道，为秦王朝统一岭南提供了重要的保证，大批粮草经水路运往岭南，有了充足的物资供应。
  • 公元前214年，即灵渠凿成通航的当年，秦兵就攻克岭南，随即设立桂林、象郡、南海3郡，将岭南正式纳入秦王朝的版图。

汉，漕渠

  • 汉漕渠，是汉武帝于元光六年(公元前129年)采纳大司农郑当时的建议，渭河之南开凿一条从长安至黄河专门运输粮食的漕渠，命齐人水工徐伯率领万人历时三年修筑而成。

三国，曹操，白沟

  • 曹操开白沟，兴建了沟通黄河和海河的第一条区间运河。
  • 《三国志·武帝纪》载：“九年春正月，济河，遏淇水入白沟以通粮道。”
  • 白沟开凿于东汉建安九年（204年），在淇水入黄河处下大枋木使成堰，遏淇水入春秋战国时期的黄河古道以成渠，以淇水、荡水、洹水为上源，利用纵贯河北平原的清河故道至今青县附近，注入滹沱河。
  • 《三国志·武帝纪》中：“公将征之，凿渠，自呼沲入泒水，名平虏渠。又从泃河口凿，入潞河，名泉州渠，以通海。”
  • 平虏渠开凿于建安十一年（206年），从今青县东北引滹沱河水北入泒(gū)水。
  • 泉州渠也开凿于建安十一年，南起泉州县（今天津市武清西南），上承潞河，下入鲍丘水，还从沟河口向东凿渠入濡水，称为新河。
  • 在曹操得胜回师，途经碣石山时，写下了充满宏伟抱负和浪漫主义情怀的不朽诗篇《观沧海》。

隋唐大运河

  • 汴河怀古（唐，皮日休）尽道隋亡为此河，至今千里赖通波。若无水殿龙舟事，共禹论功不较多。
  • 隋唐大运河以洛阳为中心，北至涿郡（今北京），南至余杭（今杭州）。
  • 隋炀帝要到扬州看琼花，为什么运河要修的北至涿郡，南至余杭？
  • 隋朝大运河，其工程分为四个部分：通济渠、邗沟、永济渠、江南运河。
  • 大业元年（605年），隋炀帝征发“河南、淮北诸郡民前后百余万”开通济渠。
  • 邗沟沟通了长江、淮河两大河流，成为隋唐大运河最早修建的一段。
  • 大业四年（608年），隋炀帝诏发河北诸郡男女百余万开永济渠，引沁水，南达于河，北通涿郡。
  • 大业六年（610年），隋炀帝下令开江南运河（即江南河），从京口（今江苏镇江）至余杭（今浙江杭州）

高句(gōu)丽(lí)

  • 高句丽（公元前37年－公元668年），是汉唐时期活跃于东北亚地区的重要民族政权，国祚七世纪有余。
  • 高句丽灭国前的常备军队在60万人左右，军事装备相当精良，兵器与中国略同。
  • 开皇十八年（598年），隋文帝命汉王杨谅、上柱国王世积为行军元帅，周罗喉为水军总管，率大军30万，分水陆两路进攻高句丽。粮草匮乏、疫病、海难，水陆两军死亡十之八九，隋军撤退。婴阳王高元上表称“辽东粪土臣元”，隋文帝罢兵，待之如初。
  • 高句丽王高元不遵籓礼，隋炀帝决定征讨高句丽。大业八年正月（612年），出征的隋朝士兵全部集中于涿郡，有军人一百一十三万三千八百，此外为其运输物资的民夫是军人数量的两倍。第一次攻高句丽之战遂以惨败而告终。
  • 大业九年（613年），二次亲征高句丽时，杨玄感造反。隋炀帝密召诸将，密令撤军。
  • 大业十年（614年），三次亲征高句丽，国内已经大乱。高句丽也困弊，遣使请降。
  • 贞观十八年（644年），唐太宗以张亮为平壤道行军大总管，率兵四万走海路。以李世勣（李勣）为辽东道行军大总管，率军六万向辽东。获胜后从容班师。以后，唐军采取了对高句丽发动骚扰性攻击的策略。
  • 唐高宗总章元年（668年），高句丽被唐朝攻灭，其遗民主要融入汉族及朝鲜半岛民族。

铁细腰

  • （唐）河堤石、铁细腰
  • （清）钦工铭文铁细腰
  • 尾燕铁，就是腰铁，因形状象两个对着的燕子尾巴，故叫燕尾铁，也称铁细腰，铁锭榫，燕尾榫、铁锁。
  • 形状是中间细，两头粗大呈三角，用白矾粘于石间，起加固作用。
  • 传说是鲁班发明的，是万榫之母。
  • 最早使用这种技术的，是隋朝的赵州桥。

裴耀卿

  • 开元二十一年（733年），裴耀卿擢黄门侍郎、同平章事，充江淮、河南转运使。
  • 关中地狭民稠，自江淮运送粮食物资困难，裴耀卿改进漕运，沿河置仓，逐级运粮，傍河凿山，开车道数十里，以避三门之险。
  • 改革漕运的成功，使得关东和江淮地区的粮食大量运抵关中，保障了关中的粮食需要，充实了关中仓储，对京城稳定，乃至全国的经济繁荣与政治稳定起到了重要作用。

苏州宝带桥

  • 宝带桥别名长桥，南北横卧于大运河和澹（dàn）台湖之间的玳玳河口，与古运河平行，是中国古代十大名桥之一，也是中国现存的古代桥梁中，最长的一座多孔石桥，是中国现存最长、保存最完整的一座大型古代连拱石桥。
  • 唐元和十一年（公元816年），宝带桥由刺史王仲舒主持建造，筹措建桥资金，王仲舒带头将自己身上的宝带捐出来，宝带桥之名由此而来，又有说因桥似宝带浮于水上而得名。
  • 清同治二年（公元1863年），英军戈登驾舰攻打苏州，拆去宝带桥桥中间大孔，酿成南面26孔连续倒塌之惨状。抗战时期，南端6孔被日军炸毁。
  • 1956年，宝带桥修葺恢复旧观，桥拱照原式样用花岗石砌成，桥面和两端桥堍也照原式样进行了修缮，并把坠入河中的4只石狮打捞出来，安装就位。

法源寺

  • 悯忠寺，法源寺。事实上，唐太宗只提了个动议，直到武则天时期才真正完成。
  • 唐太宗贞观十九年（645）及高宗上元二年（675）东征还，深悯忠义之士殁于戎事，卜斯地将建寺为之荐福。（武）则天万岁通天元年（696），追感二帝先志，起是道场，以‘悯忠’为额。《元一统志》
  • 法源寺悯忠阁。悯忠高阁，去天一握。

唐《无垢净光宝塔颂》，史思明

  • 文字虽竖排，但从左至右，而非常见的从右至左。明末清初的著名学者顾炎武认为，这可能是安禄山临时定的规矩。
  • 《无垢净光宝塔颂》仅七百六十六字，涂改的字至少达四十九字之多，有些字磨掉后，想不出新字填补，干脆留白，多达十八处。
  • 《无垢净光宝塔颂》本是用来讨好安禄山的，因安禄山初期屡战屡胜，史思明不得不表忠心。没想到安禄山死得太快，史思明只好在碑上乱涂乱改，将所有僭越的词都删去，转而吹捧唐朝。只是活儿干得太糙，一块好碑，生生被弄成千疮百孔的“百衲衣”。

茶

  • 荼和茶，陆羽之前的时代，茶写作荼，有着药的属性。
  • 斗茶、煎茶、点茶
  • 渣斗，执壶，盏，鍑(fù)，则，碾
  • 陆羽，茶风北渐，铜则，渣斗，产业文化一条龙
  • 唐代宗，竟陵积公和尚（智积和尚）
  • 唐上元初年（760年），陆羽隐居苕溪（今浙江湖州），撰《茶经》三卷，对茶的性状、品质、产地、种植、采制、烹饮、器具等皆有论述，并首次提出了“精行俭德”的茶道精神，使饮茶从物质层面升华到精神层面，该书为世界上第一部茶叶专著。
  • 陆羽以著《茶经》而闻名于世，对世界茶业发展作出了卓越贡献，被誉为“茶仙”，尊为“茶圣”，祀为“茶神”。
  • 州刺史李季卿在扬子江畔，遇见了在此考察茶事的陆羽，便相邀同船而行。李季卿闻说附近扬子江中心的南零水煮茶极佳，即令士卒驾小舟前去汲水。不料士卒于半路上将一瓶水泼洒过半，偷偷舀了岸边的江水充兑。陆羽舀尝一口，立即指出“此为近岸江中之水，非南零水。”李季卿令士卒再去取水，陆羽品尝后，才微笑道：“此乃江中心南零水也。”

湖州石家铜镜和湖州真石家铜镜

  • 宋代因避宋太祖祖父赵敬的名讳，将“镜”字改为“照”或“鉴”，故称铜镜为“照子”或“铜鉴”。

  • 宋代铸镜业由于受到政府连年用兵，推行铜禁制度，以及多种手工业新发展的冲击，铜镜大都质地轻薄，更注重实用。

  • 湖州镜是宋代最具特点的铜镜类型，始铸于北宋，兴盛于南宋，元代中衰，明代复兴。

  • 湖州镜制品大多专注实用，不尚花纹，多为素背，镜背铸有长方形印章式的商标性质铭记，标明铸镜作坊字号或工匠名，形制多作葵花形，亦有圆形和方形亚字形、桃型。

  • 湖州镜铭文详细记录产地、商号、铸造者，兼具广告与防伪功能。其铸造分官铸和私铸两大类，私铸工坊中以石家影响最大。

  • 石家是宋代著名的铸镜世家，以其世代相承、家族经营的特色闻名。

  • 铭文牌记常位于镜钮左侧或右侧，阳文楷书。

  • 铭文的内容主要是铜镜的生产地点、生产商、材质等，这是宋代宣传广告的方式在铜镜上的一种体现。

  • 类似的牌记在宋代瓷器、刻本等文物中都有出现，可以看出宋代各行各业都有私家作坊，生产品种单一，专业性很强。

  • 自主产权意识高，自己生产的东西要打上自家名号，为防止伪冒，甚至打上“真”、“真正”等字样。

  • 牌记在铜镜上的出现，改变了铜镜铭文的性质，也成为湖州镜的特征之一。

白浮泉

  • 按今天的测量，白浮泉的海拔为55米，瓮山泊的海拔为40米。
  • 白浮泉遗址，位于北京市昌平区龙山山顶，始建于元朝时期，是通惠河的水源地，建有北京历史上唯一一座以“都”字冠名的敕建龙王庙。
  • 元至元二十九年（1292年），郭守敬筑堰障水，汇西山诸水，引白浮泉水济漕，白浮泉成为通惠河上游水源，新运河被赐名通惠河。
  • 元朝时期，引白浮泉水济漕工程完成后，在龙山之巅兴建都龙王庙，并在其源头处修建九龙池。

积水潭

  • 积水潭源于古代永定河故道留下的坑洼，地下水汇集后成为水塘。
  • 积水潭曾是漕运的总码头，也是皇家的洗象池。
  • 从元代起，来自暹罗、缅甸的大象，就作为运输工具和宫廷仪仗队使用，在夏伏之日，驯养员会带领大象到积水潭洗浴。

乾隆粉彩六方套瓶

  • 清乾隆款景德镇窑外粉彩内青花镂空花果纹六方套瓶。
  • 原为一对，一只收藏于北京首都博物馆，另一只收藏于台北鸿禧美术馆。
  • 乾隆八年，督窑官唐英和助手们研制成夹层玲珑瓶九种，呈献给乾隆皇帝。六方套瓶是其中一种。
  • 1860年英法联军入侵中国期间，这件陶瓷瓶被洛克爵士掠得，后来卖给了大收藏家莫里逊氏。
  • 2000年5月2日下午2点20分，在香港苏富比，拍卖师以420万港币的底价开始叫拍，在经历了40多次的竟拍之后，北京市文物公司最终以1900万港币的价格拍得。
  • 遗憾的是，花费心血抢救国宝的秦公先生，没能亲眼看一眼国宝，在拍卖会圆满成功8天之后，因积劳成疾突发心脏病病逝。

瓷瓶的试制三难：

  • 由于瓷瓶是六方瓶，所以不能上圆盘拉坯，成型极为困难，此为一难；
  • 瓷瓶雕刻后的瓷胎由于应力改变，烧制中极易变形，此为二难；
  • 瓷瓶为内青花外粉彩套瓶，即镂空瓶中再套一瓶，其工艺极其复杂，须两次入窑烧成，成品率极低，此为三难。

十三陵水库

  • 十三陵水库的设想和修建规划。工程原定在第三个五年计划期间，由国家投资一千九百万元建设。
  • 1957年农村社会主义改造任务基本完成，广大农民建设社会主义的积极性普遍高涨。昌平区委根据群众呼声，提出了修建十三陵水库的计划。
  • 1958年1月21日，十三陵水库修建工程破土动工，最初参加建设的人员有昌平民工8000余人和义务劳动大军约2000人。虽然人们劳动热情高涨，但施工进度缓慢。
  • 为赶在汛期到来之前完成筑坝任务，1958年4月，水利电力部和北京市委决定再次增加施工力量，动员全市工、农、商、学、兵各行各业参加义务劳动，施工劳动力增至5万人。
  • 4月28日，国务院副总理贺龙、聂荣臻和北京军区司令员杨成武亲自到工地视察后提出，动员驻京部队和院校力量支持水库建设。
  • 1958年5月8日，北京市委批准水库党委由市委直接领导，指挥部按部队形式组织，从此施工人数猛增至10万。
  • 1958年5月25日下午，毛泽东、刘少奇、周恩来、朱德等中央领导人，到十三陵水库工地参加义务劳动。
  • 1958年6月30日，经过160天的修建，十三陵水库竣工。

世界遗产名录审批标准

条件凡提名列入《世界遗产名录》的文化遗产项目，必须符合下列一项或几项标准方可获得批准：

  1. 代表一种独特的艺术成就，一种创造性的天才杰作。
  2. 能在一定时期内或世界某一文化区域内，对建筑艺术、纪念物艺术、城镇规划或景观设计方面的发展产生过大影响。
  3. 能为一种已消逝的文明或文化传统提供一种独特的至少是特殊的见证。
  4. 可作为一种建筑或建筑群或景观的杰出范例，展示出人类历史上一个(或几个)重要阶段。
  5. 可作为传统的人类居住地或使用地的杰出范例，代表一种(或几种)文化，尤其在不可逆转之变化的影响下变得易于损坏。
  6. 与具有特殊普遍意义的事件或现行传统或思想或信仰或文学艺术作品有直接或实质的联系。

大运河符合的世界遗产标准

  • 标准I：代表人类创造精神的杰作。
  • 标准III：能为现存的或已消逝的文明或文化传统提供独特的或至少是特殊的见证。
  • 标准IV：是一种建筑、建筑群、技术整体或景观的杰出范例，展现历史上一个（或几个）重要发展阶段。
  • 标准VI：与具有突出的普遍意义的事件、文化传统、观点、信仰、艺术作品或文学作品有直接或实质的联系。

曲阜楷木雕如意

  • 曲阜楷雕是国家级非遗，艺人用楷木雕刻孔子像、如意、手杖等，木头纹理细密，越放越润。
  • ‌楷木（jiē mù）其实就是黄连木，主要长在曲阜孔林，是“楷模”这个词的源头‌。 -‌ 楷木不仅木头能用来雕刻和做毛笔杆，还因为孔子弟子子贡守墓的传说，成了尊师重道的象征。‌‌
  • 孔子冢上生楷，周公冢上生模。“楷模”就成了模范的代称。
  • 相传孔子去世后，弟子子贡在墓旁守了六年，亲手种下楷树并雕刻了孔子像。
