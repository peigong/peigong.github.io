# 水生生物馆(水族馆)

“水生生物馆”（水族馆）经重新改造后于2024年5月1日重新开放。作为国家自然博物馆中独一无二的活体生物展示区，该展场为公众提供沉浸式的观察体验，观众能够近距离观察众多门类水生生物的生活状态，深入了解水生生物独特的习性和行为特征。

“水生生物馆”总面积600㎡ ，展出了几十种、共上千尾活体水生生物，展馆主要由三个展区域构成，最接近入口的是大型淡水鱼类展区，展出了活体长丝𩷶、虎纹鸭嘴鲇等鱼类；在鲨鱼池和海底隧道展区，能够看到乌翅真鲨、三湖慈鲷、大眼海鲢等多种活体生物，还可以一睹珍贵的大型中华鲟标本和镇馆之宝矛尾鱼标本；第三个展区展示了我国常见的观赏鱼，包括匙吻鲟、西伯利亚鲟、金鱼、锦鲤、阿拉伯鲀、双须骨舌鱼众多珍贵的活体金鱼品种。

水生生物馆展区布局灵活巧妙，活体水生生物与生物标本兼具，体现出水生生物馆集科研价值与科普观赏于一体、为公众提供近距离观赏了解水生生物多样性窗口的目标和宗旨。
