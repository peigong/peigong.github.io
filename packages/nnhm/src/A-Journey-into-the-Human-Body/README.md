# 走进人体

经过改陈的“走进人体”展2024年10月恢复开放。该展新增8组人体组织和器官标本、10件观众互动设施，继续以图文并茂的形式和大量精美的标本为依托，系统展示人体构造的科学内容、讲述人体各器官的结构和功能。

展览除保留原有的制作精美的人体塑化标本和人体器官标本，还新增了8组珍贵标本：全身血液循环、神经系统概观、全身浅层肌肉、临床骨科手术、肌的各种形态、脾动脉、小肠动脉、肾和输尿管动脉。展览中陈列的标本不仅清晰地勾勒出人体的肌肉与经脉，而且再现了器官内部的机理和层次的质感。

“走进人体”展览新增设计制作的10个互动科普项目为：性别密码、血型由来的奥秘、食物消化之旅、小肠大作用、泌尿系统解码、肝脏大探索、五感探秘空间、血液律动、镜像骨骼互动站、奇妙的神经元八大系统。展览中的互动项目和设施进一步增强了展览的参与性和观赏性，体现展览科学性与趣味性的结合。

人类要想正确地认识和了解自身，首先应从认识自己的身躯和内部结构开始。想了解相关的人体科学知识，就应从人体的解剖结构、生理功能、器官构造等多个方面为切入点，特别是在一定的条件下，人体某个器官为何易患疾病；如何提高人体的免疫力，对各类疾病的进行预防；如何增强健康意识适度地锻炼身体，从而免于疾病的困扰等。“走进人体”展览对这些话题都一一作了介绍。
