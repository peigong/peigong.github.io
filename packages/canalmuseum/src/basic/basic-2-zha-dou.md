# 渣斗

渣斗，又名奓斗、唾壶，是中国传统工艺品和日用器皿，始见于晋代，主要用于盛放食物残渣、茶渣及漱口水。其造型源于青铜觚，形制多为喇叭口、高颈、鼓腹，瓷质最为常见，亦有银器、漆器、玻璃等材质。

明代《益部谈资》首次记载“渣斗”名称，民国《饮流斋说瓷》描述其形态为“觚之小者”。晋代青瓷渣斗为早期代表，瓷质唾壶东汉已有，三国至西晋流行撇口扁腹造型。宋代渣斗使用普及，越窑、耀州窑、官窑及景德镇窑均有烧制，常作为茶具使用，西安宋墓出土铜质渣斗内发现千年前白茶遗迹。宋季宴席常备渣斗，明清时期景德镇窑大量生产，采用郎窑红、画珐琅等装饰工艺。玻璃渣斗流行于清康熙至乾隆年间，嘉庆后品质下降。

## 基本含义

渣斗，又称唾壶、奓斗，是古代的一种卫生器具。古代贵族宴请宾客，屋内铺席，席下铺筵，为避免弄脏“筵席”，就要准备唾壶，以备客人吐口水之用。

渣斗主要用于盛放食物残渣（如肉骨、鱼刺）、茶渣及漱口水（唾吐物）。

渣斗的造型多为喇叭口、宽沿、深腹。

## 发展历程

渣斗最早被称为唾器，安徽阜阳双古堆西汉汝阴侯墓出土的此类物品上刻有“汝阴侯唾器”铭文。六朝时因其形似壶而称唾壶。据史料记载，渣斗在晋代开始使用，瓷质的较常见，比如青瓷渣斗。

唐代渣斗造型以撇口为主，腹部更加浑圆，接近盂的形状，故而又称唾盂。唐代许多窑口均烧造唾盂，如越窑、巩县窑等。唐越窑青釉瓷唾壶的釉色莹润，有“类冰类玉”之谓。

宋代渣斗口径加大超过器腹，因口部如斗而改称渣斗，并成为茶具的一种。其起源和兴盛与当时的饮茶之风密不可分。宋代许多窑场都烧制渣斗，北宋越窑、耀州窑、南宋官窑等出品都很著名。北宋早期渣斗为侈口、束颈、深腹，晚期为宽沿大敞口、束颈、圆鼓腹，南宋至元时为侈口、直颈、浅削腹。耀州窑薄胎青釉贴塑人物渣斗出土于商洛市东郊一宋墓，北宋耀州窑青釉刻花渣斗出土于陕西省西安市蓝田县吕氏家族墓。

元人笔记记载“宋季大族设席，几案间必用筋瓶、渣斗”。明清时渣斗变为大敞口，无颈部。明清时期景德镇窑大量制作渣斗，数量较多，有多种色釉和彩绘装饰。渣斗也被放置于床边和几案上，用途拓宽，材质日渐多样，有银器或漆器等。清代渣斗的质地广泛多样，除瓷质外，还有玉质、玻璃质、漆器等。清代粉彩葫芦万代五谷丰登纹渣斗展现了粉彩工艺的精湛。清中期出现透明磨花玻璃渣斗，为有模吹制后经刻面工艺琢磨加工而成，乾隆时期档案中记载的“磨花玻璃”一词与此工艺吻合。

## 基本特征

渣斗的造型随时代演变：六朝时期为壶口（又称唾壶）；唐代以撇口为主；宋代为盘口；明清时期则演变为大敞口。

其典型结构为上部呈敞口的碗状，下部承接圆腹小罐，承接口的口径有大小之分。

釉色与工艺方面，六朝时期多青釉；唐代有青釉、白釉、黄釉等，如越窑青釉有“类冰类玉”之谓；宋代则以景德镇窑的影青釉、耀州窑的青瓷等著名；明清时期釉彩更为丰富，如黄釉、粉彩等，工艺精湛，清代还出现了透明磨花玻璃等特殊材质工艺。
官窑制品常带有款识，如清代官窑渣斗底足可见“大清光绪年制”等楷书款，是判断其官窑身份及年代的重要依据。

## 文物与收藏

一件宋钧窑渣斗曾陈列于故宫武英殿古物陈列所，该渣斗约15厘米高，口径约25厘米，施蓝色乳光釉，釉面带有海棠红斑纹。

代表性文物方面，唐代渣斗如浙江省博物馆藏的唐越窑青釉瓷唾壶，通高22厘米，釉色莹润，体现了越窑“类冰类玉”的特征；泰州博物馆藏有出土于泰州西汽车站附近唐墓的巩县窑棕黄釉唾盂，造型由浅腹碗与椭圆形坛组合而成，具有典型唐代特征。宋代渣斗如商洛市博物馆（现秦岭博物馆）藏的宋代耀州窑薄胎青釉贴塑人物渣斗，1979年出土于商洛东郊宋墓，为国家一级文物，釉色呈橄榄绿，器身有壶门形开光内贴塑人物；陕西历史博物馆藏有出土于西安蓝田吕氏家族墓的北宋耀州窑青釉刻花渣斗；另有宋代景德镇窑影青釉划花葵口渣斗，白胎，影青釉，腹部饰双钩花卉纹。清代宫廷渣斗如河南博物院藏的“大清光绪年制”款黄釉奓斗，高9厘米，内外满施黄釉，为清代晚期景德镇官窑制品；故宫博物院藏有一组清中期的透明磨花玻璃渣斗（紫、绿、茶色），为有模吹制后经刻面工艺琢磨加工而成。

渣斗的重要考古发现提供了其历史演变的实物证据。例如，唐代渣斗见于江苏泰州唐墓；宋代渣斗则出土于陕西西安蓝田吕氏家族墓、陕西商洛东郊宋墓等地。

渣斗兼具实用与观赏性，是收藏市场关注的门类。其中，工艺考究的明清宫廷渣斗尤为珍贵，主要收藏于北京故宫博物院等地。2014年，一件明宣德青花“江崖海水”图渣斗在香港苏富比拍卖会上成交。晚清至民国时期的民间小渣斗存世量较大，器表装饰题材广泛，涵盖花卉、人物、山水等，在收藏市场中较为常见。

## 相关研究

相关研究主要围绕考古发现、文献记载和用途辨析展开。

陕西省西安市蓝田县吕氏家族墓出土了北宋耀州窑青釉刻花渣斗，为研究宋代渣斗的形制、使用场景及耀州窑工艺提供了重要实物依据。

宋代吴自牧《梦粱录》记载，唾盂（渣斗）是皇帝御前的执侍器物之一。元人笔记亦有“宋季大族设席，几案间必用筋瓶、渣斗”的记载，印证了其在宋代贵族生活中的广泛应用。

渣斗的用途存在演变与辨析。早期类似器物（如唾壶）用途存在争议，有卫生用品和茶具等说法。渣斗与唾壶在古人观念中有区别，主要因收纳物不同（食物残渣与唾液）。唐代中晚期以后，随着饮茶之风盛行，渣斗的作用已超出卫生用具范畴，成为一种专门的茶具，用于收纳茶事过程中产生的废弃之物（如茶渣）。
