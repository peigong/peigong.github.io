## 奉先殿

- 钟表近两百件，乾隆年间制造的最为精巧、数量最多。

### 铜镀金写字人钟

- 乾隆皇帝最爱。表盘部分，英国伦敦制表名师提摩西-威廉森专门定制。机械部分，瑞士著名的雅凯-德罗家族铸造。
- 写字人绅士，工整书写：八方向化，九土来王。
- 瑞士纳沙泰尔博物馆，收藏类似原理机械人，书写外文。
- 乾隆皇帝要求另一位会写万寿无疆的机械人，再学会写满蒙藏三种文字。机械师未能成功。

## 文华殿

- 黑色琉璃瓦
- 宁波天一阁，是江南最著名的藏书楼，藏书多达七万卷。
- 四库全书七馆：文渊、文源、文津、文溯、扬州文汇阁、文澜、镇江文宗阁

### 万历

- 1572年，万历九岁登基。内阁首辅张居正。
- 每月上朝听政九天，其他在文化殿上课学习。
- 《帝鉴图说》：张居正成立皇家教材编写小组，把治国故事配图和浅显解释，作为小皇帝万历的课本。
- 《任贤图治》，尧帝求贤若渴，任用有才能的人治理国家。
- 《览图禁杖》，唐太宗看医书说人的五脏都在背部，规定日后用刑，不得鞭打犯人的背部。
- 《敛财侈费》，唐玄宗晚年注重享乐，统治阶层生活奢华，百姓艰难，国家陷入困境。

### 文渊阁

- 明朝的文渊阁，在农民起义的战火中，付之一矩。
- 1773年，开《四库全书》馆。
- 1774年，为贮藏《四库全书》，在文华殿后面，烧毁的文渊阁原址上，设计规划重建文渊阁。
- 曾国藩在文渊阁做过四年皇家图书馆馆长助理。

## 内阁大堂

- 丞相 -> 内阁 -> 司礼监 -> 南书房 -> 军机处
- 1957年，内阁小院成为故宫博物院紫禁城学会的办公所在地。

## 武英殿

- 元大都中轴线，在武英殿东边的断虹桥上，与现中轴相距150米。
- 狮子
- 断虹桥：先有桥后有河，先于明朝皇宫。小狮子的位置高过龙，狮子在望柱，龙在栏板。
- 明朝，作为便殿，皇帝斋居和召见大臣。
- 康熙平定天下后，武英殿成了皇家刻书场，经康雍乾三朝，武英殿刻书达到极盛。
- 殿本：校勘精细，字体、刻工俱佳，用特制的墨及洁白细腻的开化纸印刷，质地精美。
- 1644年4月29日，李自成在武英殿做了一天皇帝。撤离时除武英殿以外的大部分建筑遭到焚毁。
- 清同治八年（1869年），毁于大火。

### 浴德堂

- 浴德：《礼记 儒行篇》澡身浴德
- 文华殿后院有大庖井。周礼有左庖右浴。
- 土耳其风格浴室。墙上贴素白琉璃面砖，后墙有壁炉可以烧水。
