# 陈铭枢档案

陈铭枢（1889——1965），字真如，广东合浦曲樟（今属广西）客家人，民主革命家、北伐将领，曾任民国政府军事委员、广东省政府主席、代理行政院院长等。1948年1月陈铭枢在香港与李济深等建立中国国民党革命委员会，团结各派民主人士，反对蒋介石独裁、内战政策，是民革的创始人之一。1949年6月陈铭枢到达北京参加新政治协商会议筹备会，9月出席中国人民政治协商会议，10月1日参加了开国大典。建国后，陈铭枢历任中央人民政府委员、中南行政委员会副主席、全国人民代表大会常务委员会委员、中国人民政治协商会议全国委员会常务委员、国民党革命委员会中央常务委员等职。

陈铭枢建国前后的重要档案，编为2册，共计97件（套）。其中包括1949年6月16日《新政治协商会议筹备会组织条例》等反映新政治协商会议筹备会的系列珍贵档案；1949年9月20日《中国人民政治协商会议共同纲领》（草案）及1949年9月29日《中国人民政治协商会议共同纲领》（草案）等反映中国人民政治协商会议第一届全体会议的系列珍贵档案；1949年9月中国人民政治协商会议筹备会编印《国旗图案参考资料》等反映拟定国旗、国徽、国歌的系列珍贵档案；1949年10月1日庆祝中华人民共和国中央人民政府成立典礼程序等反映开国大典的系列珍贵档案。这些系列档案是中华人民共和国成立的历史见证，不但是珍贵的文物，更具有极高的史料价值。
