# 讲解提纲

## 景山

- 景山又称煤山、镇山、万岁山、玄武山。
- 原本是一座带有风水性质的山，所以称为镇山、玄武山。
- 金大定十九年（1179年），金世宗在辽瑶屿行宫的基础上建太宁宫，并开凿了西华潭（今北海），淤泥堆成小丘。
- 13世纪中叶，元世祖忽必烈建大都，将此处辟为专供皇帝赏乐的“后苑”，将原有小丘改称“青山”。
- 明永乐十八年（1420年），在修建北京城时，用拆除元代宫殿的渣土和挖掘故宫护城河的泥土，在元代延春阁的基础上堆起一座土山，取名“万岁山”，又称“镇山”。
- 清顺治十二年（1655年），“万岁山”改名为“景山”。

## 绮望楼

- 明代建有一座五开间的大殿，称为山前殿。皇帝经常在山前殿宴请各地来京朝贺的文武官员、各部落首领及公使等，这里成为皇帝宴请宾客的宴会厅。
- 清乾隆十五年（1750年），在原建筑的基础上兴建了绮望楼。

## 寿皇殿

- 明代的寿皇殿建筑群曾偏居景山苑内东北隅。
- 清乾隆十四年（1749），寿皇殿建筑群在景山的正北重建，成为清代北京城规划形成以来中轴线上加建的重要建筑群。
- 寿皇殿曾经供奉从康熙至光绪八代帝后的画像和牌位。
- 元旦、岁暮、令节及先帝诞辰、忌辰，皇室在此恭行大礼。

## 观德殿

- 孝贤皇后
- 永恩殿、观德殿位于寿皇殿东，明代时是帝王的习射之所。
- 清代乾隆起，便改为停放帝后的灵柩之处。

## 崇祯自缢处

- 原树早已枯死伐除，现存槐树为重新栽种的，并挂有说明牌。
- 在景山东坡的歪脖老槐树下自缢身亡，时年33岁。
- 遗诏：朕凉德藐躬，上干天咎，致贼直逼京师，皆诸臣误朕。朕死，无面目祖宗，自去冠冕，以发覆面，任贼分裂，勿伤百姓一人。
- 御前太监王承恩，自缢在对面的海棠树下。

## 虬龙柏

- 明朝嘉靖年间，宫中有一只毛色特别、眉间雪白的狮子猫，名为“霜眉”。
- 猫死后，嘉靖帝哀悼，命翰林院侍读袁炜撰写悼词，其中有“化狮为龙”之句。
- 遵照嘉靖帝的旨意，猫被葬于万岁山北的一棵古树旁，并立碑“虬龙冢”以作纪念。
