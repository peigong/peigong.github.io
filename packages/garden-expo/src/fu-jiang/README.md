# 福建园·闽园

北京园博园内的闽园（即福建园）是传统展园区的“四大金刚”之一（与北京园、江苏园、广东园并列）。它不仅是北方地区规模最大的闽南红砖建筑群，更是一座将闽南、客家、妈祖文化浓缩于一体的“福建文化会客厅”。

## 基础档案

- 身份：省级展园（福建园），非城市园。
- 位置：传统展园核心区，紧邻5号门（进门左手边即是），与江苏园（忆江南）隔路相望。
- 规模：占地约1.5万平方米，是园内单体面积最大的展园之一。

## 建筑特色：红砖古厝(cuò‌)的“教科书”

闽园避开了大众熟知的土楼，选择展示更具生活气息的闽南红砖大厝(cuò‌)（民居）。

- 视觉符号：红砖白石、燕尾脊。屋顶两端像燕子尾巴一样高高翘起，极具辨识度。
- 核心建筑：和福居。这是一座占地680平方米的典型闽南古厝，采用“一埕(chéng)两进两护”布局。内部“光厅暗屋”（天井明亮、居室幽暗）的设计，以及梁枋上“添丁进财”的彩绘，都严格遵循了泉州、厦门地区的传统规制。

## 空间布局：前宅后园

全园严格遵循闽南传统“前庭后院”的格局，移步换景：

### 前庭（建筑区）

- 和福居：主厝，展示闽南人家生活场景。
- 戏台：位于古厝后院，曾是表演布袋戏、歌仔戏的民俗舞台。

### 后园（山水区）

- 方鉴池：中心大水池，池中矗立妈祖雕像，体现福建海洋文化信仰。
- 环池景观：包括南熏楼（仿厦门集美地标，展示茶文化）、风雨廊桥、闽山第一亭等。

## 文化拼图：不止闽南

虽然以闽南文化为主调（红砖厝、歌仔戏），但园内也巧妙融入了福建其他地域元素：

- 闽都文化：通过福州马头墙等细节体现。
- 客家文化：入口处的进士石旗杆广场，是客家崇文重教传统的象征。

## 与“厦门园”的区别

园博园内另有独立的厦门园（城市展园），切勿混淆：

- 闽园（福建园）：省级集锦园，融合全省文化，规模宏大，位于5号门传统展园区。
- 厦门园：市级主题园，通常侧重鼓浪屿或滨海风情，规模较小，位于城市展园区域。

## 游览攻略

- 最佳动线：从5号门入园，按江苏园（忆江南）→ 闽园 → 北京园 → 济南园的顺序游览，这是传统展园最顺路的精华路线。
- 最佳季节：四季皆宜。春季有樱花海棠，夏季池中荷花盛开，秋季南熏楼旁的银杏变黄，红砖配黄叶极为出片。
- 门票：园博园已免费，预约大门票即可进入，无需单独购票。
