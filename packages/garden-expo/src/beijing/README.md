# 北京园

北京园博园内的“北京园”位于丰台区永定河西岸，是园区内传统展园（五园之一）的核心代表。它并非现代抽象园林，而是一座严格遵循清代官式做法、再现皇家园林气韵的“园中园”。

## 设计理念与布局

核心理念：以“天、地、人和”为哲学基础，强调“天人合一”。设计上“借山得势、近水得趣”，巧妙利用地形高差，将建筑与自然山体（鹰山）融为一体。

空间结构：占地约3300平方米，采用“三进院落”布局。从入口广场到核心建筑群，空间层层递进，体现传统礼序。园内还独创了“十万景图”概念，即十个以“万”字命名的主题景观节点。

## 核心景观节点

万景门与万景千园广场：入口区域。广场设计借鉴天坛圜丘形制，地面石砖镌刻北京从春秋至“燕京八景”的历史文献，形成文化地景。

万芳亭：园内标志性建筑，采用重檐攒尖顶，梁枋保留清代官式彩画工艺，是典型的皇家制式。

万象昭辉阁：全园制高点，原型取自圆明园“茹古涵今”。登阁可俯瞰锦绣谷及永定塔，视野极佳。建筑彩绘为等级较高的旋子彩画。

知秋园（西跨院）：相对私密的庭院空间，设有月季花架，体现皇家山地园的含蓄内敛。

## 建筑与工艺特色

屋顶形制：集中展示了歇山、悬山、硬山、攒尖四种传统屋顶形式，等级分明。

彩画艺术：主要运用旋子彩画（皇家等级）和苏式彩画（人物山水题材），梁枋色彩艳丽，工艺精细。

匾额楹联：所有建筑均悬挂匾额，内容多引经据典，强化皇家园林的文化厚重感。

## 游览实用信息

地理位置：紧邻锦绣谷南侧，位于传统展园区域。从3号门（园博园地铁站对应门区）入园向东、再向北步行约10-15分钟可达。

门票政策：北京园博园目前已免费开放，北京园包含在园内，无需单独购票。

最佳体验：春季（月季花架）和秋季（看层林尽染）景致最佳。建议结合永定塔与锦绣谷规划串联游览路线。
