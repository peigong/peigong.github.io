# 青瓷到白瓷的进步

## 技术革新

青瓷向白瓷的转变是中国古代制瓷技术的重大突破，其核心在于对“铁”元素的成功控制，即通过将釉和胎中的氧化铁含量降至1%以下，从而在烧成后呈现出洁白的色泽。这一过程不仅仅是颜色的变化，更是原料筛选、窑炉技术和烧造工艺的全面革新。

以下是这一转变过程中的关键技术革新：

* 原料提纯与“化妆土”的应用这是实现白瓷的基础。早期工匠通过反复淘洗瓷土，去除杂质，降低胎体中铁的含量。对于含铁量仍较高的胎体，发明了施挂化妆土的技术，即在坯体上先涂一层细腻、洁白的高岭土泥浆，覆盖住深色胎体，再罩透明釉，从而在视觉上达到白瓷的效果。
* 窑炉结构的改进（从龙窑到馒头窑）南方青瓷多用龙窑，北方白瓷则多采用馒头窑。馒头窑具有升温快、降温也快的特点，且更容易控制烧成气氛。这种结构改变使得窑工能更精确地调节氧化焰与还原焰的转换，为白瓷所需的特定烧成环境提供了硬件支持。
* 匣钵装烧技术的普及白瓷对洁净度要求极高。为了避免明火直接接触器物导致落灰或沾染烟熏，北方窑口广泛使用了匣钵满装叠烧技术。这种单烧或套装的方式，不仅保护了釉面光洁，还提高了成品率，是白瓷能够规模化生产的关键。
* 支烧工艺的精细化为了追求器物的完美，白瓷烧造中改进了支烧具。例如隋代及北朝晚期出现的透影白瓷，其胎壁极薄，透光性强，这要求支钉必须细小且耐高温，以避免在精细的胎体上留下明显疤痕，体现了成型与装烧工艺的高度结合。

这一技术体系的成熟，最终在北齐至隋代完成了由青到白的跨越，并在唐代形成了著名的“南青北白”格局。白瓷的出现不仅是审美的变迁，更为后世釉下彩绘（如青花瓷）的发展奠定了必要的白色底色基础，极大地拓宽了中国陶瓷的艺术表现力。

## 核心工序

青瓷瓷土无法直接“淘洗”成白瓷瓷土，而是通过原料优选、多级水力淘洗、磁选除铁、配方重构四大核心工序，将高铁着色瓷土提纯为低铁高铝的瓷胎。

### 一、 原料优选与预处理

- 低铁高铝矿源：优先选用原生低铁高岭土、优质瓷石，奠定白胎基础。
- 物理破碎：通过水碓舂碎、球磨、过筛，将瓷石或硬质黏土粉碎为细粉，释放包裹型铁杂质。
- 水力分级：利用水流将粗颗粒与细颗粒分离，剔除粗砂及重矿物。

### 二、 多级淘洗与除铁（核心提纯）

- 沉淀淘洗：加水搅拌，利用密度差让重矿物（铁、钛等）沉底弃除，取上层悬浮细浆。
- 磁选除铁：传统工艺采用磁石/永久磁铁，现代工业采用电磁除铁机、永磁棒、高梯度磁选机，吸附游离铁屑与磁性矿物。
- 多级过滤：通过绢筛、细筛反复过滤，截留肉眼可见杂质与粗颗粒。

## 古代除铁工艺

古代无现代磁选机时，主要靠原料优选、水力淘洗、过筛、磁石除铁、防污染管理等组合工艺，将瓷土中的铁杂质降至最低。

### 一、源头优选与预处理

- 低铁矿源：优先开采深层瓷土、优质高岭土、瓷石，从源头降低基础铁含量。
- 物理破碎：通过水碓舂打、石碾粉碎、水浸化解，将块状瓷土化作细浆，释放包裹型铁杂质。

### 二、水力淘洗与沉淀（核心物理除铁）

- 水簸法：利用水流与密度差，让重质含铁矿物、粗砂率先沉底弃除，细泥悬浮转入下一池。
- 多级沉淀：采用多池串联，反复转移细泥浆，使铁质与粗颗粒逐级沉降分离。
- 陈腐助分：泥浆长时间静置陈腐，促进铁质聚集沉淀，提升后续分离效率。

### 三、过筛与磁石除铁

- 细密过筛：使用竹筛、细绢、铁锅形筛具，拦截粗颗粒砂石与杂质。
- 磁石吸附：将磁石置于筛具或泥浆流经处，吸附游离铁屑与磁性矿物。
- 反复除铁：在泥浆过筛、转移、过滤环节多次放置磁石，层层截留铁杂质。

### 四、全流程防污染管理

- 工具替代：开采、粉碎、运输环节避免使用铁器，改用木、石、竹、铜等非铁工具。
- 环境管控：原料加工区保持洁净，严防外部铁锈、尘土二次混入。

### 五、辅助工艺与审美转化

- 化妆土遮盖：胎体铁含量偏高时，在胎体表面施一层低铁细泥浆，遮盖灰暗胎色。
- 还原焰烧制：控制窑炉气氛，将三氧化二铁还原为低色度氧化亚铁，弱化铁呈色，形成青白、牙白等温润质感。

## 判断铁含量是否达标

判断铁含量是否达标，主要依靠实验室化学分析进行定量检测，并结合感官观察、试烧验证进行综合评估。

### 一、 实验室定量检测（最准确）

- 化学分析法：通过化学试剂滴定，精确测定瓷土中氧化铁（ \text{Fe}_2\text{O}_3 ）的具体百分比。
- 仪器分析法：使用X射线荧光光谱仪（XRF）等设备，快速无损地获取铁元素的精确含量数据。

### 二、 感官与物理观察

- 泥浆颜色：观察淘洗后的泥浆或滤饼颜色。含铁量越低，颜色越接近纯白色或灰白色；若偏黄、偏灰或偏红，则表明铁杂质依然较高。
- 断面检查：观察干燥后坯体的断面，肉眼或借助放大镜查看是否有深色斑点（铁点）。

### 三、 试烧验证（最终标准）

- 制作试片：取少量淘洗后的泥浆制成薄片或小样。
- 高温烧制：放入窑炉中经高温烧成。
- 成品比色：观察烧后试片的白度。若白度达到预期且无杂色斑点，即判定为合格。

### 四、 不同白瓷标准的铁含量参考

- 普通日用白瓷： \text{Fe}_2\text{O}_3  含量通常需控制在 0.5% - 0.8% 以下。
- 精细高档白瓷： \text{Fe}_2\text{O}_3  含量需降至 0.3% - 0.5% 甚至更低。
- 极致透明白瓷：对原料纯度要求极高，需配合精选高岭土和严格除铁工艺。

## 龙泉窑白胎瓷土

龙泉窑白胎瓷土并非单一高白高岭土，而是本地瓷石、高岭土、紫金土按特定比例调配而成，核心为瓷石+高岭土+少量紫金土的复合配方 。

### 一、瓷土来源与成分

1. 本地瓷石（主体）
   - 成分：富含石英，含绢云母、少量高岭土/长石 。
   - 特点：高硅（SiO₂ 69%–78%）、低铝（Al₂O₃ 14%–22%），提供烧结基础 。
2. 高岭土/硬质黏土（强化）
   - 作用：提升Al₂O₃含量，增强胎体强度与耐火度，防高温变形 。
3. 紫金土（调色助熔）
   - 成分：含石英、长石、含铁云母等，Fe₂O₃含量3%–5%（高者达15%）。
   - 作用：白胎中少量添加，微调胎色与釉色衬托，非黑胎主因 。
4. 胎色成因
   - 白胎：Fe₂O₃约1.9%–2.5%，TiO₂约0.2%–0.5%，呈白/灰白/白中泛青 。
   - 黑胎：高紫金土比例，高铁高钛，呈灰黑/紫黑 。

### 二、加工技术（七步精制）

1. 选矿配料
   - 按目标胎色，称取瓷石、高岭土、紫金土 。
2. 风化陈放
   - 露天堆放3–6个月，提高原料活性与可塑性 。
3. 粉碎磨浆
   - 传统：水碓/石碾粗碎→细碎 。
   - 现代：颚破/轮碾/球磨，控制颗粒分布 。
4. 淘洗分级
   - 加水搅浆→过筛（去粗杂）→沉淀池静置 。
   - 取上层/中层细浆，去底层粗砂铁质，反复沉淀提纯度 。
5. 压榨脱水
   - 沉淀泥块→布袋/压滤脱水→软硬适中的泥饼 。
6. 陈腐醒泥
   - 泥饼密封存放数月，改善可塑性，减少干燥收缩 。
7. 练泥除气
   - 传统：人工/牛脚踩踏半月，挤气泡，匀密度 。
   - 现代：真空练泥，提升成品率 。

### 三、核心特点与实证

1. 技术特点
   - 配方：瓷石+高岭土+少量紫金土复合体系 。
   - 加工：依赖精细淘洗与陈腐练泥，非单一高白原料 。
   - 胎色：受控于紫金土比例与烧成气氛，非单纯“越白越好” 。
2. 实证材料
   - 科技检测：EDXRF/XRF成分分析、显微结构观测 。
   - 考古报告：大窑、金村、溪口等窑址发掘标本 。
   - 文献数据：上海硅酸盐研究所等机构化学组成与显微测试 。
