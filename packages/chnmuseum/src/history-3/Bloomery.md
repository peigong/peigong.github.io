# 块炼铁

块炼铁是人类历史上最古老的冶铁技术，也被称为“海绵铁”或“锻铁”。它是在固态下还原铁矿石，而非像现代高炉那样将铁完全熔化。下面为你拆解其核心原理、炼制过程及技术特点。

## 什么是块炼铁？

块炼铁是铁矿石在 800–1000℃ 的相对低温下，通过木炭固态还原得到的产物。由于温度低于铁的熔点（约1538℃），铁不会熔化成液体，而是形成一种疏松多孔、夹杂大量矿渣的海绵状固体，故得名“海绵铁”。

成分特性：含碳量极低（<0.3%），属于熟铁。质地柔软、延展性好，但不能铸造，只能通过锻打成型。

历史地位：起源于约公元前1500年的西亚，是欧洲直至中世纪以前的主要炼铁方式。中国早期（如春秋时期）也有使用，但很快发展出了更先进的生铁冶炼技术。

## 块炼铁炼制技术与全过程

块炼铁的生产属于间歇式操作，一炉一炉地生产，无法像高炉那样连续出铁。

1. 炉窑建造：地炉或碗式炉

早期工匠通常在平地或山麓挖坑砌筑小型炉窑，结构非常简单：

炉型：多为“碗式炉”或竖炉，直径约0.7–1米，高约1–1.5米。

材料：用石头或粘土垒砌，配备简易的皮囊或风箱鼓风。

2. 装料与还原：固态反应

配料：将高品位铁矿石（块状）与木炭分层装入炉中。

鼓风加热：点燃木炭并鼓风，将温度维持在1000℃左右。在此温度下，矿石中的氧化铁被一氧化碳（CO）还原为金属铁：

\text{Fe}_2\text{O}_3 + 3\text{CO} \rightarrow 2\text{Fe} + 3\text{CO}_2 

产物状态：铁以固态颗粒形式析出，与未还原的脉石（SiO₂等）混合在一起，形成一团糊状的“铁疙瘩”。

3. 出炉与锻打：挤渣成材

这是块炼铁最费工费时的环节：

破炉取铁：冶炼结束后，炉内产物是固态的，必须破坏炉体或扒开炉膛才能取出海绵铁块。

热锻挤渣：趁热对海绵铁进行反复锤打。目的是将夹杂在铁颗粒之间的液态炉渣挤出，同时将疏松的铁块锻打成致密的铁材。未经锻打的块炼铁强度极低，无法直接使用。

4. 进阶：块炼渗碳钢

单纯的块炼铁太软。古代工匠通过在炭火中反复加热锻打，让碳渗入铁中，提高含碳量，从而制成块炼渗碳钢，用于制作更锋利的兵器。

## 技术特点与局限性

块炼铁技术生铁冶炼（对比）温度低温固态还原（~1000℃）高温液态冶炼（>1150℃）产物海绵状熟铁，含渣多液态生铁，可铸造效率极低，间歇式，单炉仅几公斤高，连续出铁，产量大加工必须锻打，费时费力可直接铸造成型 

主要缺点：燃料消耗巨大、铁回收率低（炉渣中含铁量高达70%–80%）、无法生产大型铸件。这也是它最终被生铁和炒钢技术取代的原因。

## 考古与实物证据

中国：甘肃临潭陈旗磨沟遗址出土的铁条（约公元前14世纪）是目前中国最早的块炼铁实物；春秋时期的玉柄铁剑（河南三门峡虢国墓地）也采用了此技术。

世界：该技术通过西亚传至欧洲，统治了西方冶铁业近2000年。

块炼铁虽然原始，但它是人类开启铁器时代的钥匙，其“固态还原+锻打”的原理也是现代粉末冶金的技术雏形。
