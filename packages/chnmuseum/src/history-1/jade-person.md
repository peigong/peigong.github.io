# 跽坐玉人

河南安阳殷墟妇好墓（商王武丁时期）的跽坐玉人。玉人左侧后方斜插的物件，在考古界被称为“宽柄器”（或柄形器）。

关于这个“宽柄器”的具体性质和用途，目前学术界尚无绝对定论，但主要有以下几种主流推测：

1. 王权与军权的象征（微缩玉钺/权杖）：这是最主流的观点。该宽柄器上端呈卷云状内卷，与商代妇好墓出土的真实兵器（如玉钺、青铜戚）形制高度吻合。在商代，“国之大事，在祀与戎”，玉人背后插着“钺”，象征着墓主人妇好作为女战神所拥有的军事统帅权和王权。
2. 礼仪用具（玉笏/珽的雏形）：有学者考证，这可能是古代插在腰间的玉礼器“珽”或“荼”（即后世玉笏的前身）。《荀子》中记载“天子御珽，诸侯御荼”，贵族在上朝或祭祀时将其插于腰带上，作为身份和等级的象征（即“搢绅”）。
3. 通神法器（巫觋祭祀用品）：商代盛行占卜和祭祀，妇好本人也是大祭司。这件宽柄器也可能是一种沟通人神的法器，用于在祭祖、祈福等仪式中使用。
4. 装饰或图腾象征（龙/蛇尾或鸟羽）：也有部分观点认为它并非实用器，而是纯粹的装饰物，或者是某种图腾（如龙尾、蛇尾，亦或与商族“天命玄鸟”崇拜相关的鸟羽）的抽象化表现，象征着人兽结合的神秘力量。

总的来说，这个宽柄器并非日常实用工具，而是商代高等级贵族身份、权力与宗教信仰的集中体现。这件玉人本身也被许多专家认为是妇好本人的雕像，具有极高的历史与艺术价值。
